This one is my first freelancing Flutter Project for client. I already around into flutter. So i have to learn to build and design beautiful real-life app from the ground up.

  • To build complete android & iOS Uber Clone apps (passenger) with Flutter and Firebase
  • To create, remove, update and delete records in Firebase Database
  • Tow to use Google Location and Map services in your Apps with Flutter
  • To find place address with location coordinates using Google Geocoding API
  • Tdvanced state management concepts with the Provider Package
  • To create and design beautiful User Interfaces for android and iOS using Flutter
